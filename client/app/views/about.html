About Drug Reactions Explained

As medical terms get harder and harder to understand, Drug Reactions Explained (DRE) provides clear definitions to the medical terms used to describe the reactions patients have experienced when taking certain drugs.

This project was inspired and developed for openFDA.

How to use DRE

Using the search bar, type a brand name or drug to begin. DRE provides the top 6 reaction cards associated with the searched and an analytical dashboard.

Reaction Cards

The top 6 reactions associated with the searched term are populated into interactive cards. The reaction term is placed onto the front of the card. Clicking the card, flips it to the back where clear definitions are provided for better understanding.

Card Actions

Users can add definitions if none are defined. User also have the ability to upvote and downvote definitions depending on the quality of the term definition.

Dashboard

The dashboard provides an analytic view into openFDA's drug API. Users will gain insight into top manudactures, brands, gender and more reports.

Filters

Use the sidebar to select filter actions on the dashboard.

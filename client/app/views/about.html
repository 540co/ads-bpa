About Drug Reactions Explained

As medical terms get harder and harder to understand, Drug Reactions Explained (DRE) provides clear definitions to the medical terms used to describe the reactions patients have experienced when taking certain drugs.

This project was inspired and developed for openFDA.

How to use DRE

Using the search bar, type a brand name or drug to begin. DRE provides the top 6 reaction cards associated with the searched term and an analytical dashboard.

Reaction Cards

The top 6 reactions associated with the searched term are populated into interactive cards. The reaction term is placed onto the front of the card. Clicking the card, flips it to the back where clear definitions are provided for better understanding.

Card Actions

- Upvote a term definition

- Downvote a term definition

- Add a new term definition

- Flip card

Dashboard

The dashboard provides an analytic view into openFDA's drug API and can view high level information in context of reported adverse effects of a drug/substance.

Categories

Use the sidebar to add or remove categories to refine the reults of a given searched term.

Note: The results are in context of an adverse report which may contain one or more drugs. Information on the dashboard is not always in context of a single drug.
